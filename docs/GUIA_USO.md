\# 📖 Guia de Uso



\## Como executar o banco de dados



\### Passo 1: Abrir MySQL Workbench

\- Clique no ícone do MySQL Workbench

\- Faça login na conexão padrão



\### Passo 2: Criar nova aba SQL

\- Clique em \*\*File\*\* → \*\*New Query Tab\*\*

\- Ou aperte \*\*Ctrl+T\*\*



\### Passo 3: Executar schema.sql

\- Abra o arquivo `database/schema.sql` com Bloco de Notas

\- Copie TODO o conteúdo

\- Volte para MySQL Workbench

\- Cole no editor SQL

\- Clique no botão ⚡ \*\*Execute\*\*

\- Ou aperte \*\*Ctrl+Enter\*\*



\### Passo 4: Verificar se funcionou

Execute este comando:



```sql

SHOW TABLES;

```



Você deve ver 10 tabelas listadas.



\### Passo 5: Inserir dados de teste (opcional)

\- Abra `database/sample\_data.sql`

\- Copie o conteúdo

\- Cole em uma nova aba SQL no MySQL Workbench

\- Execute



\## Pronto! 🎉



Seu banco de dados está funcionando!
